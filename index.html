AgeCheckls.js Demo
jQuery Plugin for Checking user age

AgeCheckls.js will automatically prompt to check your age. It will redirect back to this page on success. You can change the minimum age by selecting from the select options, and re-trigger by clicking the big button below.

Oh, the plugin is also responsive - so resize the viewport. Aitte?
